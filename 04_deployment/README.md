# 04 Deployment

## Overview
Deploy LLM models and chains to Databricks Model Serving using various approaches and frameworks.

## Learning Objectives
- Deploy models using Databricks Python SDK
- Use Databricks Agents framework for deployment
- Deploy PyFunc models with environment variables
- Deploy model-as-code and LlamaIndex chains

## Prerequisites
- Registered models in Unity Catalog (from `02_rag_chains/` or `03_agents/`)
- Model Serving endpoint capacity
- Databricks workspace tokens

## Notebooks
- `00_deploy_pysdk.py` - Deployment with Databricks SDK
- `01_deploy_dbagent.py` - Deployment with Databricks Agents
- `02_deploy_modelfile.py` - Deploy model-as-code chains
- `03_deploy_llama_index.py` - Deploy LlamaIndex models
- `04_deploy_advanced_agent.py` - Deploy advanced agent applications
