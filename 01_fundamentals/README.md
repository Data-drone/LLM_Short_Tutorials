# 01 Fundamentals

## Overview
Introduction to LLM operations on Databricks, covering basic prompting, MLflow logging, and document processing pipelines.

## Learning Objectives
- Setup and query Databricks LLM endpoints using OpenAI client
- Log prompts and responses with MLflow 3.0 autologging
- Parse and chunk documents using LangChain and LlamaIndex
- Create and populate Databricks Vector Search indexes

## Prerequisites
- Databricks workspace with serving endpoints enabled
- Unity Catalog access

## Notebooks
- `00_prompting_and_logging.py` - Basic prompting and MLflow experiment tracking
- `01_parsing_chunking_langchain.py` - Document processing with LangChain
- `02_parsing_chunking_llama_index.py` - Document processing with LlamaIndex
