# 00 Setup

## Overview
Utility scripts for workspace setup, authentication, and data preparation.

## Learning Objectives
- Generate and manage Databricks secrets
- Create and configure model serving endpoints
- Scrape web content for document sources
- Build datasets for RAG applications

## Prerequisites
- Databricks workspace admin or appropriate permissions
- Unity Catalog access

## Scripts
- `00_generate_secrets.py` - Create Databricks tokens and secrets
- `01_create_endpoints.py` - Provision model serving endpoints
- `02_webscraping.py` - Web scraping utilities for data collection
- `03_building_dataset.py` - Dataset preparation and upload
